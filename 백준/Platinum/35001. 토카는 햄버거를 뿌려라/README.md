### 문제 설명

<p>토카는 카툰컵에 참가한 $N$명 중 $M$명에게 햄버거를 추첨해서 주고자 한다. 각 참가자는 $1$번부터 $N$까지 서로 다른 정수로 번호가 매겨져 있고, $i$번 참가자가 푼 문제의 개수는 $A_i$이다.</p>

<p>토카는 다음 과정을 통해 햄버거를 받을 사람을 추첨하려 한다.</p>

<ul>
<li>아직 뽑히지 않은 사람들 중에서 한 명을 무작위로 뽑는다. 각 사람이 뽑힐 확률은 자신이 푼 문제의 개수의 제곱에 비례한다.</li>
<li>$M$명이 뽑힐 때까지 이 과정을 반복한다.</li>
</ul>

<p>$1$번 참가자인 한별이는 자신이 햄버거를 받을 확률이 얼마나 되는지 궁금해졌다. 한별이가 햄버거를 받을 확률을 구해보자!</p>

### 입력 

 <p>첫 번째 줄에 대회의 참가자 수를 나타내는 정수 $N$ $(1 \leq N \leq 20)$과 햄버거를 받을 사람의 수를 나타내는 정수 $M$ $(1 \leq M \leq N)$이 주어진다.</p>

<p>두 번째 줄에 각 참가자가 푼 문제의 개수를 나타내는 $N$개의 정수 $A_1, \cdots, A_N$ $(1 \leq A_i \leq 100)$이 주어진다.</p>

### 출력 

 <p>한별이가 햄버거를 받을 확률이 기약분수 $a \over b$일 때 $a \times b^{-1} \, \bmod \, 1\,000\,000\,007$을 출력한다. $1\,000\,000\,007$은 소수이며, $b^{-1}$은 $b$의 모듈러 곱셈에 대한 역원이다.</p>

<p>정답이 유리수인 경우만 주어진다. 정답의 분모가 $1\,000\,000\,007$의 배수인 입력은 주어지지 않는다.</p>
